## 🚀 Getting Started

### 1️⃣ Clone the Repository

```sh
git clone https://github.com/your-username/sumith-portfolio.git
cd sumith-portfolio
```

### 2️⃣ Install Dependencies

```sh
npm install
```

### 3️⃣ Run the Development Server

```sh
npm run dev
```

### 4️⃣ Build for Production

```sh
npm run build
```
